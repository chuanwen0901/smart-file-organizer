# 📁 智能文件整理器

## 功能概述
自动整理电脑文件，解决文件混乱问题。支持批量操作，智能分类，重复检测。

## 主要功能

### 1. 智能分类
- **按类型分类**：文档、图片、视频、音频、压缩包等
- **按时间分类**：按年/月/日整理
- **按项目分类**：根据内容智能分组

### 2. 批量重命名
- **序列重命名**：file_001, file_002...
- **时间戳重命名**：20240301_文档.pdf
- **内容重命名**：从文件内容提取关键词

### 3. 重复文件检测
- **MD5校验**：精确检测重复文件
- **相似度检测**：相似图片/文档
- **智能处理**：保留最新/最大版本

### 4. 清理优化
- **空文件夹清理**：删除空文件夹
- **临时文件清理**：清理系统临时文件
- **大文件查找**：找出占用空间的大文件

## 使用场景

### 个人使用

## 安装要求

### 系统要求
- Windows 10/11
- Python 3.8+
- 足够的磁盘空间

### Python依赖
- 标准库：os, shutil, hashlib, datetime
- 可选：Pillow（图片处理）, PyPDF2（PDF处理）

## 快速开始

### 基本使用
```bash
# 整理指定文件夹
openclaw file-organizer organize --path "C:\Users\风\Downloads"

# 批量重命名
openclaw file-organizer rename --pattern "photo_{num}.jpg"

# 查找重复文件
openclaw file-organizer find-duplicates
```

### 配置选项
```yaml
# 配置文件示例
file_organizer:
  rules:
    documents: ['.pdf', '.doc', '.docx', '.txt']
    images: ['.jpg', '.png', '.gif', '.bmp']
    videos: ['.mp4', '.avi', '.mov', '.mkv']
  actions:
    move_to_categories: true
    create_backup: true
    ask_before_delete: true
```

## 详细功能

### 分类规则
支持自定义分类规则：
```python
# 自定义分类
categories = {
    '工作文档': ['.docx', '.xlsx', '.pptx'],
    '设计素材': ['.psd', '.ai', '.sketch'],
    '代码文件': ['.py', '.js', '.java', '.cpp']
}
```

### 重命名模式
多种重命名模式：
1. **序列模式**：file_001, file_002...
2. **时间模式**：YYYYMMDD_HHMMSS
3. **内容模式**：从文件提取关键词
4. **混合模式**：时间+序列

### 智能处理
- **自动跳过系统文件**
- **处理权限问题**
- **进度显示和日志**
- **错误恢复机制**

## 高级功能

### 1. 智能标签
基于文件内容自动打标签：
- 文档：合同、报告、简历、发票
- 图片：风景、人物、截图、设计
- 视频：教程、会议、娱乐、家庭

### 2. 统计分析
生成整理报告：
- 文件类型分布
- 空间占用分析
- 重复文件统计
- 整理效果评估

### 3. 计划任务
支持定时自动整理：
```bash
# 每天凌晨整理下载文件夹
openclaw file-organizer schedule --time "02:00" --path "C:\Downloads"
```
