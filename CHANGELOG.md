# 更新日志

智能文件整理器的所有显著更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-03-01

### 新增
- **核心整理功能**
  - 智能文件分类（按类型、时间、内容）
  - 批量重命名（序列、时间戳、内容模式）
  - 重复文件检测（MD5哈希校验）
  - 空文件夹清理
  
- **用户界面**
  - 命令行界面（CLI）
  - Web图形界面（GUI）
  - 进度显示和实时反馈
  
- **配置系统**
  - YAML配置文件支持
  - 自定义分类规则
  - 行为设置（备份、确认等）
  
- **计划任务**
  - 定时自动整理
  - 批量处理多个文件夹
  - 报告生成和导出
  
- **安全特性**
  - 操作前备份
  - 操作日志和撤销
  - 权限检查和保护

### 技术特性
- **跨平台支持**：Windows、macOS、Linux
- **性能优化**：批量处理、内存优化
- **扩展性**：模块化设计，易于扩展
- **文档完整**：详细的使用指南和API文档

### 系统要求
- Python 3.8+
- 4GB RAM（推荐8GB+）
- 100MB磁盘空间

## 已知问题
- 某些特殊字符文件名可能处理异常
- 网络驱动器上的文件可能权限不足
- 超大文件（>100MB）处理较慢
