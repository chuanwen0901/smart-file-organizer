**智能文件整理器**是一款基于OpenClaw的AI助手技能，能够自动分类、重命名、整理电脑文件，彻底解决文件混乱问题！

## 🚀 快速开始

### 安装只需一步：
```bash
# Linux/macOS
curl -O https://raw.githubusercontent.com/cao-organizer/file-organizer/main/install.sh
bash install.sh

# Windows
# 下载 install.bat 并双击运行
```

### 使用示例：
```bash
# 整理下载文件夹
openclaw file-organizer organize --path "~/Downloads"

# 批量重命名照片
openclaw file-organizer rename --pattern "vacation_{num:03d}.jpg"

# 查找重复文档
openclaw file-organizer find-duplicates --path "~/Documents"

# 每天自动整理
openclaw file-organizer schedule --time "02:00" --path "~/Downloads"
```
